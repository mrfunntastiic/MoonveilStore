# MoonveilStore

## Project Overview
MoonveilStore is an online store application built with the latest technologies to provide a seamless shopping experience. Featuring a clean UI and robust backend, it allows users to browse, search, and purchase products effortlessly.

## Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/mrfunntastiic/MoonveilStore.git
   ```
2. Navigate into the project directory:
   ```bash
   cd MoonveilStore
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Available Scripts
- `npm start`: Runs the app in development mode.
- `npm test`: Launches the test runner.
- `npm build`: Builds the app for production.

## Configuration
- Configure environment variables in a `.env` file in the root directory:
  ```bash
  REACT_APP_API_URL='https://api.moonveilstore.com'
  ```
